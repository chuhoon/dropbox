### 느낀점

1. select box와 option은 JavaScript를 사용하지 않으면 커스텀이 어렵다.
2. option의 커스터마이징을 자유롭게 하고 싶다면 select 태그를 사용하지 않고 직접 만드는 것도 괜찮다. CSS, JavaScript 직접 짜야한다.
